📡 Dispatch - Get Started in Seconds

Deploy PHPCop to your project and start patrolling:

🌍 Global Installation

Install once, use everywhere

composer global require hfryan/php-cop
phpcop scan

📦 Per-Project Installation

Add to your project dependencies

composer require --dev hfryan/php-cop
php vendor/bin/phpcop.php scan

📥 PHAR Download

Zero dependencies, perfect for CI/CD

wget https://github.com/hfryan/php-cop/releases/latest/download/phpcop.phar
php phpcop.phar scan

⚙️ GitHub Actions

Automated security patrols in CI/CD

- name: PHPCop Security Scan
  uses: hfryan/php-cop@main
  with:
    fail-on: 'high'
    comment-pr: true

Common Patrol Commands

phpcop scan Basic security patrol
phpcop scan --format=html Generate HTML report
phpcop scan --fail-on=moderate Strict enforcement mode
phpcop scan --exclude-dev Production-only scan
